Device Mapping Rules

Rules that match measurement file paths to a device in the library. Checked in order of priority (higher first), then longer pattern first. Scope limits a rule to one ingestion pipeline (baselines, sc, irradiation, avalanche) or applies it to all. Pattern type substring does case-insensitive containment; regex is matched against the upper-cased file path.
